# Sistema_PointOfSale
Objetivo Final: Proyecto Funcional
## Presentación del proyecto
En este proyecto tecnológico crearemos un sistema de punto de venta que consiste en tres vistas separadas.
Una es la de los clientes, se dividen por mesa y cada mesa tiene su propia interfaz. El cliente o grupo de clientes
puede hacer una cantidad indefinida de pedido con una cantidad indefinida de cartas (osea los distintos tipos de 
bebida, platos, postres, etc...) la unica manera de saber que grupo hizo tal pedido es por el numero de mesa.
La vista de los cocineros se encuentra en la cocina, donde reciben los datos de todos los pedidos pendientes.
Los pedidos pendientes divididos por numero de mesa han de ser concluidos con el apretar de un boton. Y si entran
mas pedido por la misma mesa simplemente aparece otra notificacion del pedido del correspondiente numero de mesa.
La vista del financiero funciona como un consultador de SQL, con opciones como mostrar el importe total de tal mesa,
la cantidad de tiempo trascurrido desde el primer pedido hasta el ultimo, entre otros.
